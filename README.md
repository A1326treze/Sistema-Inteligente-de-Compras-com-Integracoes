# Sistema-Inteligente-de-Compras-com-Integracoes
Um sistema de planilha inteligente, via GoogleSheets, com integração de AppScript, LookerStudio, AppSheet e PowerBI

## AppScript
Engranagens que fazem a planilha funcionar, adicionando Menus personalizados, sistema de gradientes e separadores de itens para Matriz de pesquisa de preços, histórico que transforma a Matriz em tabela e vice-versa com opção de save em lote, desfazer, refazer e verificação de duplicatas, Topbar flutuante opcional para facilitar navegação, limpenza da Matriz de pesquisa para facilitar a inclusão de novos preços rápidos, ocultação de coluna para mostrar apenas as lojas essenciais, padronização automatica de escrita para evitar possiveis bugs (espaços depois de palavras), Filtro de setores para mostrar apenas o essencial e sincronização da Matriz de preço com AppSheet
## LookerStudio
Gráficos que facilitam a visualização de preços e suas flutuações sobre diferentes itens, pacotes e marcas, sendo possivel verificar possiveis tendencias de preços
## AppSheet
Versão mais compacta e essencial da planilha, integrando os principais Scripts e facilitando a captação de preços pessoalmente. Conta com Syncronização de dados planilha <> AppSheet
## Power BI
Em construção. Alternativa ao LookerStudio, devido a sua maior gama de opções.
